Für Multi-User-Sync wird eine kostenlose Firebase-Datenbank benötigt.
Einmalige Einrichtung (~5 Min).

Firebase einrichten
1
console.firebase.google.com öffnen → „Projekt erstellen" → Namen eingeben → Google Analytics deaktivieren → Erstellen
2
Im linken Menü: Erstellen → Realtime Database → „Datenbank erstellen" → Region wählen → „Im Testmodus starten" (30 Tage offen, reicht für den Test)
3
Die angezeigte Datenbank-URL kopieren. Sie sieht so aus:
https://PROJEKTNAME-default-rtdb.firebaseio.com
Wird lokal im Browser gespeichert – nicht geteilt.
